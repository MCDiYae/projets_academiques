Objectifs

L’objectif de la formation est de Former des développeurs d’applications web et mobiles (Bac +5) maîtrisant les technologies les plus demandées dans le marché d’emploi au niveau national et international. Plus spécifiquement, l’objectif est des former des programmeurs capable de:

Développer des applications frontend et backend.

Développer des applications mobiles.

Développer en cloud.

Développer des bases de données pour le web.

Maîtriser les processus et les outils de gestion de projet.

Héberger et administrer des sites web.

Public visé

Etudiants ou cadres d’entreprises qui souhaitent acquérir des compétences dans des technologies web et mobile avancées: Titulaires d’un diplôme de Bac+3 ou plus en informatique, mathématiques ou physique ou autres spécialités jugées équivalentes du secteur public ou privé.

Programme de la formation

La formation est répartie sur 4 semestres et chaque semestre est constitué de 4 modules:

Semestre 1

  • M1 : Initiation web: HTML – CSS – PHP
    Ce module concerne l’exploration des différentes fonctionnalités du HTML et du CSS pour le développement des pages web statiques ainsi que l’initiation au PHP pour le développement des pages web dynamiques.
  • M2 : Développement Orienté Objet & Java-JEE (Spring Framework)
    • Programmation orientée Objet en Java
    • Bases du langage Java
    • Principes de la programmation objet et UML
    • Spring framework – Spring Web – Spring Data – Spring Security – Microservices et Spring Cloud
  • M3 : Programmation Python pour le développement web
    Initier l’étudiant à se familiariser avec les concepts de python et apprendre à concevoir et à créer et maintenir un site web dynamique en utilisant le Framework Django.
  • M4 : Javascript avancé et Angular
    • Respecter le modèle de développement en couches
    • Maitriser le développement des composants réutilisables
    • Communiquer avec l’application BackEnd
    • Réutiliser les éléments Angular déjà développés

Semestre 2

  • M5 : React JS, React Native et PWA
    Le module React commence par reprendre les fondamentaux de JavaScript utiles au framework, ensuite les bases de React, les composants, Redux, le système de centralisation des données et des actions, le routage, JSX, puis les spécificités de React Native.
  • M6 : Développement des sites web avec les CMS
    Développement des sites web informationnels et boutiques en ligne par l’utilisation des outils élaborés : CMS (Content Management System).
  • M7 : Cloud Computing
    Ce module est focalisé sur la technologie Cloud Computing. Il fournit une introduction et la définition de domaine, il présente les caractéristiques essentielles et les principaux avantages du Cloud Computing. Il décrit également les principaux modèles de services et les principaux modèles de déploiement.
  • M8 : Base de données orientées Web
    Ce module concerne les bases de données dans les environnements web. De leur modélisation à leur traitement en passant par les questions de standards qui orientent le choix entre les bases de données relationnelles et les base de données NoSql.

Semestre 3

  • M9 : Internet of Things
    • Programmation mobile et Web
    • Internet des objets (IoT)
  • M10 : ServiceNow
    ServiceNow fournit une solution « Software as a service » qui centralise l’ensemble des processus de l’entreprise et transforme les anciennes méthodes de travail en workflows numériques.
    Ce module couvre tout ce qui est nécessaire pour maîtriser les scripts sur la plateforme ServiceNow, ainsi que le fonctionnement des API ServiceNow.
  • M11 : Infographie pour le web
    Photoshop et Illustrator sont des logiciels incontournables pour la conception et la réalisation d’interfaces web répondant à des critères techniques spécifiques. Dans ce module, nous abordons la préparation des logos, des visuels et des interfaces graphiques, ainsi que la réalisation des bannières publicitaires pour le web et le mobile et la préparation des chartes graphiques des sites web.
  • M12 : Soft Skills – Gestion de Projets – Méthodes de Test
    • Soft-Skills : Compétences sociales, personnelles et méthodologiques.
    • Gestion de Projet : Contraintes, Intervenants, phases de gestion, cycle de vie et méthodes de gestion.
    • Méthodes de Test.

Semestre 4

  • Stage professionnel
    Un stage professionnel de 4 mois, avec un mémoire à préparer et une soutenance devant un jury.
    En raison du caractère professionnel de ce DCESS, le stage en entreprise occupe une place importante dans cette formation et l’évaluation des apprenants.
